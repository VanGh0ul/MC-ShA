Остановился на 2.1.1.3


Форма авторизации:
	Ввод эл. адрес
	Ввод пароля
	Кнопка вход в систему
	Надпись (подчеркнуто) "Регистрация"

	Кнопка вход в систему
		Вход в систему
		Если удачно
			Ресет
			окно просмотра товаров

	Надпись Регистрация
		Ресет
		окно регистрации

	Метод вход в систему
		непривязанный пользователь при входе в систему будет получать сообщение в виде диалогового окна о том, что у него нет привязки и для полноценной работы с системой он должен создать организацию в личном кабинете, либо связаться с сотрудниками организации в котором он работает для привязки (диалоговое окно Windows);

	Ввод ФИО
	Ввод эл. адреса
	Ввод пароля
	Ввод повтор пароля
	Радиобатон создать бизнес
	Радиобатон работник
	Кнопка Назад
	Кнопка Далее

	Кнопка Назад
		Ресет
		окно авторизации

	Кнопка Далее

			Переход на форму создания организации

		Иначе
			Создание нового пользователя
			Форма аторизации.Вход в систему
			Ресет
			Переход на форму просмотра товаров 

	Метод Создание нового пользователя
	Метод Ресет

Форма создания организации
	Ввод названия
	Ввод эл. адреса
	Ввод телефона
	Выбор картинки
	Вывод выбранной картинки
	Кнопка Отмена
	Кнопка Далее
	
	Кнопка Отмена
		Если открыта из формы регистрации
			Ресет
			Переход на форму регистрации

		Если открыта из формы личного кабинета пользователя
			Ресет
			Переход на форму личного кабинета пользователя

	Кнопка Далее
		Форма регистрации.Создание нового пользователя
		Форма авторизации.Вход в систему
		Создание новой организации
		/* Выборка информации о организации */
		/* Привязка пользователя к организации */
		Переход на форму просмотра товаров


	Метод Ресет
	Метод Создание новой организации

Форма личного кабинета организации

	Привязанный пользователь:

			Переход на форму заказов

				Надпись ФИО
				Если не пользователь, что смотрит
					Кнопка Отвязать

		Надпись "Код привязки пользователя"
		Текстовое поле Код пользователя
		Кнопка Добавить сотрудника


	Администратор:
		Если Венера
			Нет списка иходных заказов
			Нет списка входных заказов
